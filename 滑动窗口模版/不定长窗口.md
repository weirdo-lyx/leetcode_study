leetcode 3. 无重复字符的最长子串
给定一个字符串 s ，请你找出其中不含有重复字符的 最长 子串 的长度。

 

示例 1:

输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。注意 "bca" 和 "cab" 也是正确答案

class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        left = ans = 0
        cnt = defaultdict(int)
        for right in range(len(s)):
        
            while cnt[s[right]] > 1: //循环看是不是因为最左边导致的重复
                cnt[s[left]] -= 1
                left += 1   
            cnt[s[right]] += 1 //把字符加进来
            ans = max(right-left+1,ans) 
        return ans

不定长的核心思路就是这个while循环遍历看看左边是不是导致了条件不满足然后移动左指针。注意要先把字符加进来，而不是放在while下面

当然这道题可以简单点用set
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        seen = set()
        left = 0
        max_l = 0
        for i in range(len(s)):
            while s[i] in seen:
                seen.remove(s[left])
                left += 1
            max_l = max(max_l,i-left+1)
            seen.add(s[i])
        return max_l

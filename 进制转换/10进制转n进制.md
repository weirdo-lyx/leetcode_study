def convert(num,n):
    if num == 0:
        return "0"
    pat = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ"
    res = ""
    while num != 0:
        remain = num % n
        res = pat[remain]+ res
        num = num // n
    return res


思路是用除以基数得到余数 然后把余数反写 注意是res = pat[remain]+ res
而不是 res = res + pat[remain]这样保证了反向拼接
